
# Crypto Trading Bot Paneli

Bu proje, BNB/USDT için fiyatları grafik üzerinde gösteren ve RSI-7/14/24 göstergelerini içeren bir paneldir.

## Kurulum

1. Gerekli modülleri yükleyin:
```
npm install
```

2. Sunucuyu başlatın:
```
npm start
```

## Render İçin

Dockerfile varsa Render kullanabilir.
